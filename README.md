
# Todo List 50

A small Elm application built for architectural exploration.

This project intentionally focuses on:

- Strong domain modeling
- Explicit state transitions
- Functional transformation helpers
- Clean separation between Model, Update, and View

It implements basic CRUD functionality:

- Create todos (form submission + Enter key)
- Read todos
- Toggle completion status
- Delete todos

The goal of this project is not production readiness, but practicing clean functional architecture in Elm.
